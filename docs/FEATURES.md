# DyslexAI Features

## User-Facing Pages

| Page | Route | Description |
|------|-------|-------------|
| **Landing** | `/` | Welcome; sign up / log in links |
| **Signup** | `/signup` | Create account |
| **Login** | `/login` | Sign in |
| **Dashboard** | `/dashboard` | Metrics, chart, recent OCR history |
| **Workspace** | `/workspace` | Upload image, OCR, view raw/corrected text and layers |
| **History** | `/history` | List of OCR runs; approve/edit/reject |
| **Exercises** | `/exercises` | Student picker, exercise types (typing, handwriting, tracing) |
| **Game Mode** | `/game` | Gamified exercise flow |
| **Students** | `/students` | Manage students |
| **Settings** | `/settings` | User preferences |
| **About** | `/about` | Project info |
| **Help** | `/help` | Help content |
| **Privacy** | `/privacy` | Privacy policy |
| **Terms** | `/terms` | Terms of use |
| **Library** | `/library` | Content library |

---

## Key Features

- **OCR:** Upload handwriting image → raw text + corrected text + per-layer view
- **Auth:** Signup, login, JWT, protected routes
- **Dashboard:** User-scoped metrics and history
- **Exercises:** Word typing, sentence typing, handwriting, tracing
- **Tracing canvas:** On-screen letter/word tracing
- **Dark theme:** Consistent UI
