Theme CSS Variables (from :root)

Open DevTools on the running app (http://localhost:4173 or 5173), select <html>, and run in Console:

const s = getComputedStyle(document.documentElement);
['--color-primary','--color-corrected','--color-background','--spacing-md','--radius-md'].forEach(k => {
  console.log(k + ': ' + s.getPropertyValue(k).trim());
});

Expected output:

--color-primary: #308ce8
--color-corrected: #fff59d
--color-background: #f5f5f5
--spacing-md: 16px
--radius-md: 12px
